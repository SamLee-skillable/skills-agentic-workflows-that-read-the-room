## Latest GitHub Updates

- **GitHub Copilot SDK is now generally available (June 2, 2026):** Developers can embed Copilot's agentic engine directly into their own apps via a stable API. The SDK ships in six languages (Node.js/TypeScript, Python, Go, .NET, Rust, and Java) and supports custom tools, system prompt customization, and OpenTelemetry tracing. Available to all Copilot subscribers. ([changelog](https://github.blog/changelog/2026-06-02-copilot-sdk-is-now-generally-available/))
- **Copilot CLI gains voice input, rubber duck mode, and prompt scheduling (June 2, 2026):** A major CLI update adds a redesigned terminal UI, a "rubber duck" second-opinion mode for reviewing code plans, hands-free voice input, and `/every`/`/after` commands for scheduling recurring or delayed prompts. ([changelog](https://github.blog/changelog/2026-06-02-copilot-cli-improved-ui-rubber-duck-prompt-scheduling-and-voice-input/))
- **New enterprise installation API in public preview (May 13, 2026):** GitHub App developers can now check whether their app is installed on a given enterprise and retrieve its installation ID without paginating all installations — bringing parity with org, repo, and user installation APIs. ([changelog](https://github.blog/changelog/2026-05-13-new-enterprise-installation-api-now-in-public-preview/))
